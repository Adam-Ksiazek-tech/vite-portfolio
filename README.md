# Vite Portfolio

Moje osobiste portfolio stworzone w **React**, **TypeScript** i **Vite**, z wykorzystaniem **TailwindCSS** i **i18next** do obsługi wielojęzyczności.

---

## Technologie

- **Frontend:** React 19, React DOM
- **Budowanie:** Vite
- **Styling:** TailwindCSS
- **Internacjonalizacja:** i18next + react-i18next
- **TypeScript** dla typów i bezpieczeństwa
- **Linting:** ESLint z pluginami dla React i hooks
- **Animacje:** motion
- **Ikony:** lucide-react
